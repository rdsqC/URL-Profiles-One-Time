## どんな物なのか
これは情報を入力してプロフィールのURLを作成し、第三者が開くことで内容を確認できます。
## 仕組み
URLのあとに
?(変数名)=(変数値)
と書くことでjavascirptからURLの変数を取得できます。詳しくは後の出典をご覧ください。
